# Jungle Heritage Resort

This is the source code for the Jungle Heritage Resort & Spa website.

## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation
1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
